## Architecture

Glyx MCP consists of two main components:

### glyx-python-sdk (Core Framework)

The SDK provides the foundational framework for:
- Agent configuration and execution
- Orchestration primitives
- Memory management
- Pipeline workflows

**Location**: `packages/sdk/`
**Package**: [glyx-python-sdk on TestPyPI](https://test.pypi.org/project/glyx-python-sdk/)

### glyx-ai (MCP Server)

The FastMCP/FastAPI server providing:
- MCP tools for Claude Code integration
- REST API endpoints
- WebSocket support
- GitHub and Linear webhooks

**Location**: `src/glyx/`

## Use Cases

- **Autonomous Coding**: Deploy AI agents to implement features, review code, and run tests
- **Task Automation**: Create smart tasks from selected text with AI-powered generation
- **Knowledge Management**: Build a searchable memory of project decisions and patterns
- **Integration Hub**: Connect GitHub, Linear, and other services through webhooks
